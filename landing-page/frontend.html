Đối tượng học viên

Đam mê thiết kế, lập trình ứng dụng web, thương mại điện tử, landing page.

Có tư duy Visual Thinking - tư duy mỹ thuật trực quan.

Sinh viên, người đi làm trái nghành muốn trở thành lập trình web.

Sinh viên CNTT năm cuối làm đồ án tốt nghiệp web front end.

Sinh viên, nhân viên marketing, thiết kế đồ hoạ muốn bổ sung kỹ năng lập trình web.

Nội dung chính

Thiết kế giao diện UI và phân tích , tối ưu trải nghiệm người dùng UI.

Cắt Photoshop file, Figma xuất ra HTML, CSS.

Lập trình JavaScript, jQuery kết hợp CSS, Bootstrap để thiết kế responsive web.

Lập trình JavaScript ES 6, Node.js dựng REST API.

Hoàn thành 3 đồ án môn học: thiết kế giao diện, lập trình web site hoàn chỉnh và ứng dụng Single Page App ReactJS.

Lộ trình học

Thiết kế UI/UX cho giao diện

Định hướng sản phẩm - kỹ thuật lấy yêu cầu, phân tích nhu cầu thiết kế sản phẩm của khách hàng. Tìm hiểu UX -UI tổng quát.

Ứng dụng UX trong sản phẩm chi tiết. Tư duy thiết kế - ai cũng có thể!

Nắm chắc các quy tắc thiết kế cơ bản: kỹ thuật bố cục (layout), phối màu (coloring), sử dụng font chữ (typography)...Tìm hiểu về GUI (Graphical User Interfaces).

Đánh giá, so sánh thiết kế xấu vs tốt từng điểm cụ thể. Hiện đã có rất nhiều mẫu thiết kế dự án từ các khóa học trước đây.

Web nâng cao

Chuyển từ thiết kế giao diện Photoshop, SketchApp, Figma sang mã HTML, CSS, JavaScript của website.

Lập trình hiệu ứng tương tác với JavaScript tăng mức độ tương tác, hấp dẫn người dùng

Lập trình web site responsive xem trên desktop, tablet và smart phone

Đồ án xây dựng web site chuyên nghiệp hoàn chỉnh

Nodes REST/JWT

Lập trình Javascript ES6, arrow function, promise, async, await.

Lập trình Node.js, Express framework dựng dynamic web application

Xây dựng REST API cung cấp dữ liệu trả về cho ứng dụng di động và ứng dụng web

Bảo mật REST bằng JWT token

ReactJS

Lập trình ứng dụng Single Page App với React.js, JavaScript framework nhu cầu tuyển lớn nhất hiện nay.

Xử lý tương tác.
