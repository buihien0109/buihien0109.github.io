vd :
var user = {
    name : "Trương Minh Thúy",
    age : 25,
    email : 'support@techmaster.vn'
};
=> name,age,email

2. Viết hàm để lấy danh sách các value của object
vd :
var user = {
    name : "Trương Minh Thúy",
    age : 25,
    email : 'support@techmaster.vn'
};
=> 'Trương Minh Thúy',25,'support@techmaster.vn'

3. Viết hàm kiểm tra key có tồn tại trong Object không
vd :
var user = {
    name : "Trương Minh Thúy",
    age : 25,
    email : 'support@techmaster.vn'
};
=> checkKey('name') => true
checkKey('study') => false

    email : 'support@techmaster.vn'
};
=> getLengthObject(user) => 3
